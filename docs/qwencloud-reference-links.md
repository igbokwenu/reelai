Key submission implications as of July 5, 2026:

- Submit to Track 2: AI Showrunner.
- Show QwenCloud API usage in a public repo.
- Include Alibaba Cloud deployment proof, including a screenshot or recording of running resources.
- Include an architecture diagram and a roughly 3-minute public demo video.
- Ensure a code file visibly includes a QwenCloud base URL such as `https://dashscope-intl.aliyuncs.com/compatible-mode/v1`.
